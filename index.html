3 API 키 입력 (Apps Script)

상단 탭 '확장 프로그램 → Apps Script'로 이동합니다.

index.html 파일에서 Ctrl+F를 눌러 api 찾아 " " 사이에 본인의 API 키를 입력 후 저장합니다.

4 웹 앱 배포
